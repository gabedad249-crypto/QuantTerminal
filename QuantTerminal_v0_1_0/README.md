# Quant Terminal v0.1.0-alpha

Professional FVG research and paper trading platform foundation.

## Run
Double-click `run_quant_terminal.bat`.

## What this version includes
- PySide6 professional dark UI shell
- Auto-created folders
- Settings file
- Logging
- Coinbase BTC-USD websocket with REST fallback
- Live candle builder
- Basic candlestick chart with zoom/pan
- Basic FVG box detection
- Paper trading account skeleton with live P/L
