# Quant Terminal v0.2.1 Alpha

Focus: smoother workflow and next-step fixes.

## Changed
- Fixed `VERSION` import issue.
- Chart no longer shows ENTRY/STOP/TARGET unless you click **Suggest Buy-In From Chart**.
- AI panel no longer resets scroll every tick.
- Suggested buy-in now automatically recommends side, entry, stop, target, and RR from recent price action.
- Open paper trade now requires an active suggested/manual plan.
- Removed chart snap-back behavior caused by forced fit/reset calls.

## Run
Double-click `run_quant_terminal.bat`.
